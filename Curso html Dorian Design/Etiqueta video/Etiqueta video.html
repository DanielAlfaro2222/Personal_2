Etiqueta video

Esta etiqueta como su nombre lo indica sirve para agregar un video en nuestra pagina.

Atributos

  • Autoplay => Se usa para indicarle al navegador que una vez que se entre o recargue la pagina, el audio automaticamente se reproducira.
  • Mute => Se usa para decirle al navegador que una vez que empiece a reproducirse el audio, no tendra sonido o lo tendra apagado.
  • loop => Se usa como bucle para que la cancion cuando se acabe vuelva a repetirse.
  • controls => Se usa para proporcionar controles al audio, para pausar la musica, adelantar, bajar el volumen etc...
  • poster => Se usa para darle un fondo al video antes de que el usuario lo reproduzca, cuando lo reproduzca se desaparecera el fondo y el video se mostrara normal.
