Estructura basica de un formulario

  • Form: Esta etiqueta lo que va realizar es funcionar como un contenedor, todo lo que este adentro de esta etiqueta va a ser el formulario.
  • Label: Esta etiqueta va a ser el nombre del campo en el formulario.
  • Input: Esta etiqueta es donde el usuario pondra sus datos o llenara el formulario.
  • Button: Esta etiqueta va a ser el boton que permitira enviar el formulario.
