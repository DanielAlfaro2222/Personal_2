Inputs extras

Tipo color

Este tipo de input se utiliza para seleccionar un color, cuando el usuario le de clic aparecera una paleta de color para seleccionar un color.

Tipo number

Este tipo sirve para que el usuario digite solo numeros, si tratamos de escribir una letra no nos dejara o no la escribira en el recuadro.

Tipo range

Este tipo de input lo que hara es dibujar una linea horizontal en la cual podremos mediante atributos elegir el minimo valor del rango, el valor maximo y de cuanto en cuanto se va a ir moviendo este rango.

Tipo reset

Este tipo de input lo que hara es crear un boton el cual su funcion sera resetear el formulario sin recargar la pagina.

Tipo text

Este tipo de input creara un recuadro donde el usuario podra ingresar texto o cualquier caracter, este tipo de input es el valor por defecto en caso de no colocar nada o no especificar que tipo de input es.
