Button vs input type button

Definicion de button

La etiqueta button sirve para enviar los datos que el usuario alla puesto en el formulario.

Ejemplo

							
								<form>
									<input type = "text">
									<button>Enviar</button>
								</form>

Definicion de input type button

Este tipo de input a diferencia de la etiqueta button no nos servira para enviar los datos que se pongan en el formulario, mas bien lo usaremos para otras cosas como por ejemplo una funcionalidad de javascript la cual se va activar cuando presionen el boton.Posdata: el input type button por defecto va a aparecer sin nombre, entonces tendremos que colocar el atributo value para nombrar el boton.

Ejemplo

							
								<form>
									<input type = "text">
									<input type = "button" value = "Enviar">
								</form>

Podremos ver que si ingresamos texto en el input y damos clic en enviar no pasara nada, en cambia si hacemos lo mismo pero en el ejemplo de el button lo que pasara es que se recarga la pagina
