Input tipo checkbox

Este tipo de input es igual a los input de tipo radio pero con la diferencia que este tipo de input se usa para que el usuario pueda escoger mas de una opcion.

Ejemplo

¿Que equipos conoces?

Atributos

Atributo name

Este tipo de atributo es para asignarle un nombre al campo, como todas las opciones son de la misma pregunta deben llevar el mismo name, pero con diferente value.

Atributo value

Este atributo es para asignarle un valor a cada opcion de la pregunta, cada valor del value deben ser distintos ya que dependiendo que opcion marque el usuario se enviara el nombre y valor correspondiente a la opcion.

Atributo checked

Este atributo es para marcar una opcion por defecto, a diferencia de el tipo de input radio podremos marcar varias opciones.
