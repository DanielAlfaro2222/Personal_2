Input para moviles

Estos tipos de input los usaremos para mejorar la experiencia de usuario cuando entren a nuestro sitio por celular, ya que dependiendo cada input el teclado saldra de manera diferente. Por ejemplo si en un formulario necesitamos que el usuario ingrese la contraseña, pues lo correcto es usar el input de tipo password y asi dependiendo de los campos del formulario varia el tipo de input que usaremos.

Input type number

Este tipo de input sirve para que el usuario ingrese solo numeros como por ejemplo un numero telefonico.

Input type search

Este tipo de input es un recuadro en el cual el usuario podra realizar una busqueda, como por ejemplo el recuadro que al final tiene una lupa en youtube ese es un input de type search.

Input type password

Este tipo de input es un recuadro en el cual la persona podra ingresar la contraseña, las letras o caracteres que ingrese el usuario se veran como asteriscos.Un ejemplo de esto podria ser en un login de una aplicacion la cual los caracteres que ingrese el usuario se veran como asteriscos para proteger lo que escriba el usuario.

Input type

Este tipo de input se usa como un cuadro de texto para ingresar un correo, por ejemplo en un login cuando necesitemos que el usuario ingrese su correo.

Input type

Este type de input se usa para ingresar una url o direccion.
