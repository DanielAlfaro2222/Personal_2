Input de tipo radio

Este tipo de input se utiliza para cuando tenemos varias opciones y queremos que el usuario selecione una.Por ejemplo en un formulario donde tenemos un campo que pregunta por el genero de una persona en este caso es importante usar este tipo de input.

Ejemplo

Atributos

Atributo name

Este atributo lo usaremos para identificar o decirle al navegador que el usuario solo puede escoger una opcion, para lograr esto todas las opciones deben de tener el mismo valor del name.Tambien sirve para identificar el nombre del campo para que cuando se envie los datos aparezca el nombre y el valor del campo que se envio.

Atributo value

Este atributo lo usaremos para identificar o decirle al navegador que opcion escogio el usuario por ejemplo en un formulario donde se pregunta el genero, nosotros podremos colocar el nombre de la opcion y el valor para que cuando se envie el formulario aparezca el nombre y el valor del campo.

Ejemplo

                        
                            <form>
                                <label>
                                    Sexo:
                                    <input type="radio" name = "genero" value = "Hombre">
                                    Hombre
                                    <input type="radio" name = "genero" value = "Mujer">
                                    Mujer
                                </label>
                            </form>

Atributo checked

Este atributo sirve para definir una opcion por defecto, este atributo es booleano.Como consejo es bueno no poner el valor de este atributo true o false lo recomendado es dejarlo sin valor ya que si colocamos este atributo en una opcion ya estariamos indicando que por ser la unica opcion el valor va a ser true.
