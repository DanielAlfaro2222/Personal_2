Input de seleccion datalist

Este tipo de input es parecido al tipo de input select ya que es una lista de opciones que tiene el usuario para escoger, con la particularidad que el usuario puede buscar la opcion que necesite osea agregar un filtro para buscar la opcion que mas el guste.

    • <input type = "list" list = "">
    • <datalist>

      • <option value = ""></option>
      • <option value = ""></option>
      • <option value = ""></option>
      • <option value = ""></option>
    • </datalist>
  • </form>
