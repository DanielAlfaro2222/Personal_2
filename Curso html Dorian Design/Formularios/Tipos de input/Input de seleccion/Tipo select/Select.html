TIpo select

Este tipo sirve para crear una lista despegable de opciones las cuales el usuario podra escoger una o varias opciones dependiendo lo que deseemos realizar.

Estructura basica

  • <select name = "">
    • <option value = "">Nombre<option>
  • </select>

Agrupar opciones

Para poder facilitar al usuario el poder escoger opciones, usando la etiqueta optgroup para agrupar opciones referentes a un tema en particular, para poder nombrar ese grupo usaremos el atributo label.

Atributos

Atributo name

Este atributo sirve para nombrar el campo de seleccion, siempre este atributo va a ir en el select nunca en las opciones.

Atributo value

Este atributo sirve para colocar un valor en cada opcion que este dentro del select, este atributo unicamente va a estar presente en la etiqueta option.
