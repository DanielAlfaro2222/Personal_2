Meter

Esta eiqueta sirve para representar un rango conocido, ademas maneja ciertos atributos min y max(representan los limites inferior y superior), low y high(representa limites desde las margenes de bajo y alto), optimum(representa el valor optimo) y value que representa el valor del rango.Esta etiqueta la podremos usar en ejemplos como el consumo total de un disco duro, el uso de una memoria ram, etc...

Progress

Esta etiqueta sirve para representar el progreso de una tarea.Esta etiqueta maneja dos atributos el primero es max(representa el valor maximo) y value (representa el progreso de la tarea).
