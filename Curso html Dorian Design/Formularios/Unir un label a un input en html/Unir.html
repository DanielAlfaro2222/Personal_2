Como unir un label a un input en un formulario

Lo que haremos para unir un label a un input es:

  1. En el input le pondremos un atributo id para identificarlo, este id debe ser unico.
  2. En el label utilizaremos un atributo for en el cual pondremos el nombre del id del input.
  3. Listo ya atamos un label a un input en un formulario.

Codigo de Ejemplo

Formas de hacerlo

                
Forma #1

                <form action = "#" method = "POST">
                    <label for = "Nombre">Nombre</label>
                    <input type = "text" id = "Nombre">
                    <button>Enviar</button>
                </form>
            
                
Forma #2

                <form action = "#" method = "POST">
                    <label>
                        <input type = "text">
                        <button>Enviar</button>
                    </label>
                </form>
