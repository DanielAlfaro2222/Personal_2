Etiqueta em

La eiqueta em sirve para darle importancia a un texto, pero si se compara con la etiqueta strong es mucho mas importante la etiqueta strong que la etiqueta em

Etiqueta strong

La etiqueta strong sirve para darle mayor importancia a un texto, es mas importante que usar la etiqueta em

Etiqueta small

La etiqueta small es para colocar texto menos importante que el texto normal.

colocar un salto de linea en el texto

Etiqueta wbr

La etiqueta wbr es para colocar un salto de linea opcional por si el texto es muy largo
