Tablas

En HTML podremos manejar o crear tablas, las cuales iran dentro de un contenedor llamado table. A continuacion podremos ver la estructura de una tabla en html:

  • Etiqueta table que va a funcionar como un contendor
    • Etiqueta thead para el encabezado de la tabla
      • Etiqueta tr para agregar una nueva fila en la tabla
        • Etiqueta th para definir el titulo de cada columna(posdata el numero de columnas es proporcional al numero de th que esten presentes en la tabla
    • Etiqueta tbody para el contenido o el cuerpo de la tabla
      • Etiqueta tr para una nueva fila
        • Etiqueta td para ir creando las columnas
