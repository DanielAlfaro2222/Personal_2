Etiqueta a para enlaces

Esta etiqueta podremos realizar de una manera mas facil la navegacion de la pagina, podremos realizar navegacion interna y navegacion externa.Ademas esta etiqueta tendra dos atributos:

Atributos

Atributo href

Con este atributo podremos colocar la ruta donde se encuentra la pagina a la cual querramos que el link nos lleve cuando le demos click.

Atributo target

Este atributo basicamente lo usaremos para indicar que accion realizara la pagina cuando le den click al enlace.Por ejemplo, podremos especificar si queremos que el link se abra en la misma pestaña del navegador donde nos encontramos o en una nueva pestaña, dependiendo que queramos realizar usaremos una u otra.

Tipos de navegacion

Navegacion interna

Este tipo de navegacion la usaremos cuando querramos navegar en secciones que se encuentren en nuestra pagina.

Navegacion externa

Este tipo de navegacion la usaremos cuando querramos redirigir la navegacion a una pagina externa de nuestra pagina, solo en ese caso usaremos la navegacion externa.
