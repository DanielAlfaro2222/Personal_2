Iframes en HTML5

Un iframe basicamente es tener una pagina web externa dentro de nuestra pagina, hay que usar la menor cantidad de iframes posibles ya que como nos estamos trayendo una pagina, la velocidad de carga de la pagina se podria ver afectada, es por eso que se recomienda usar pocos iframes.

Atributos basicos

  • src => Se usa para indicar la url donde se encuentra el sitio alojado
  • width => Se usa para indicar el ancho del iframe.
  • height => Se usa para indicar el largo del iframe.
