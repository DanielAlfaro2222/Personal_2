Contenido embebido

Es el contenido externo que usamos en nuestra pagina, ademas estos archivos son los que mas peso(tamaño) tienen.

Tipos mas conocidos

  • Imagenes
  • Audio
  • Video
  • Iframes

Imagenes

Tipos

  1. Vectoriales
    • svg (se recomienda usar siempre que se pueda)
  2. Mapa de bits
    • jpg
    • png (se recomienda usar si se necesita transparencia)
    • gif (se recomienda usar si se necesita una imagen animada)
    • webp (se recomienda usar ya que el tamaño de estas es el que menos pesa)
