Estructura basica de una tabla

A continuacion veremos la estructura basica de una tabla en html:

  1. Abertura Etiqueta table
    1. Etiqueta caption para el titulo
    2. Etiqueta thead para el encabezado de la tabla
      • Etiqueta tr para una nueva fila
        • Etiqueta th para ir creando las columnas de la fila
    3. Etiqueta tbody para el contenido de la tabla
      • Etiqueta tr para una nueva fila
        • Etiqueta td para ir creando las columnas de la fila
    4. Etiqueta tfoot para el pie de pagina de la tabla
      • Aqui aplica lo mismo tr para una nueva fila y td para ir llenado las columnas de la fila
  2. Cierre etiqueta table
