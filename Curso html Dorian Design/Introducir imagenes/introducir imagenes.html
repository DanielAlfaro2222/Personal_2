Introducir imagenes

Para introducir cualquier tipo de imagen ya sea de tipo de mapa de bits o vectoriales, usaremos la etiqueta img.

Atributos

  • src: Se usa para indicar la direccion donde se encuentra la imagen.
  • alt: Se usa para colocar texto alternativo, el cual agrega una descripcion de la imagen en caso de que no cargue o que el usuario use asistencia para personas invidentes.
