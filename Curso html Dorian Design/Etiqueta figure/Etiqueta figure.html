Etiqueta figure

Esta etiqueta se usa para representar un contenido independiente, aunque se relacion con el flujo principal, si lo movemos o lo eliminamos el flujo principal seguiria teniendo sentido, en pocas palabras su posicion es independiente del contenido prinicipal y no va a afectar el flujo principal. Se usa para representar una imagen, una ilustración, un diagrama, un fragmento de código, o un esquema al que se hace referencia en el texto principal.

Poner un titulo o pie de foto a la etiqueta figure

Para colocar un titulo o pie de foto al contenido, usaremos la etiqueta figcaption. Normalmente esta etiqueta se pone despues de la foto.
