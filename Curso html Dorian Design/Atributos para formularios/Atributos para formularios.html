Atributos para formularios

Atributo placeholder

Esta atributo sirve para colocar un texto descriptivo adentro de un campo para que el usuario sepa que debe poner en ese campo del formulario, apenas el usuario escriba en el campo este texto descriptivo se eliminara.

Ejemplo atributo placeholder

Atributo requeried

Este atributo sirve para indicarle al navegador que un campo es obligatorio llenarlo o que ese campo sencillamente no se puede dejar vacio.

Ejemplo atributo requeried

Atributo readonly

Este atributo sirve para colocar un campo de un formulario para solo lectura, no se podra editar.

Ejemplo atributo readonly

Atributo disabled

Este atributo funciona igual que el atributo readonly, con la unica diferencia que este campo se desactivara, en pocas palabras el formulario va a ignorar este campo y no lo va a enviar.

Ejemplo atributo disabled

Atributo selected

Este atributo funciona para poner un elemento de la lista del select por defecto, parecido al atributo checked de la etiqueta radio o checkbox.

Ejemplo atributo selected

Atributo min y max

Este atributo sirve para colocar un valor minimo y un valor maximo para la etiquete number, la cual establecera un rango para colocar numeros.

Ejemplo atributo min y max

Atributo minlenght y maxlenght

Este atributo sirve para establecer un rango de cuantas letras minimo debe ingresar el usuario y cuantas maximo podra ingresar el usuario.

Ejemplo minlenght y maxlenght

Atributo autofocus

Este atributo sirve para definir que campo del formulario apenas entra el usuario tenga el cursor automaticamente, este atributo solo lo podra tener un campo del formulario.

Ejemplo atributo legend
