Enlaces: Rutas absolutas y relativas

Rutas absolutas

Para empezar este tipo de rutas necesitan manejar un protocolo ya sea el protocolo http o https y son unicos en la red, ademas son mas usadas en enlaces de navegacion externa.Por ejemplo:

http://wwww.google.com

Rutas relativas

Este tipo de ruta primero no necesitan o no usan protocolos, se les llama relativas ya que se pueden encontrar en el mismo nivel de ruta, un nivel mas antes o un nivel superior con respecto a la carpeta en la que nos encontremos.

Si queremos acceder al nivel mas superior posible usaremos slash en la ruta del href, ademas no necesitamos colocar el nombre de archivo ya que por defecto el navegador buscara el archivo principal de html en este caso index.html.Por ejemplo:

"/"

Si queremos acceder a un nivel anterior de la ruta en la que nos encontramos usaremos dos puntos y slash.Por ejemplo:

"../Carpeta_nivel_anterior/index.html"

Tipos de navegacion

Navegacion interna

Este tipo de navegacion es para referir links que se redireccionaran dentro de la propia pagina.Ademas, este tipo de navegacion podra contar con una ruta absoluta o relativa segun sea el caso.

Navegacion externa

Este tipo de navegacion lo usaremos para referirnos a enlaces o links que al abrirlos nos redireccionaran a una parte externa de la pagina, ademas este tipo de navegacion contara con una ruta absoluta o relativa segun sea el caso.
