Etiqueta picture

Esta etiqueta se usa para especificar un numero especifico de opciones, en las cuales se especificaran con la etiqueta source y usando el atributo media podremos especificar la condicion que leeera el navegador, tambien aparte de las opciones (etiqueta source) tendremos una etiqueta img que se usa para especificar que si el navegador no escoge alguna de las opciones que estan dentro de la etiqueta source, pues cargara lo que haya dentro de la etiqueta img.

Source

La etiqueta source representara una opcion, cada opcion que nosotros pongamos se representara a traves de un source.

Atributos

  • srcset => indica la ruta donde se encuentra el archivo, puede ser una ruta relativa o una ruta absoluta.
  • media => indica la condicion mediante el cual cargara el archivo, si esta condicion se cumple, se cargara el archivo.

Ejemplo

Podremos desde el inspector de elementos achicar o agrandar el tamaño de la pantalla y asi poder ver como dependiendo del tamaño cargara una imagen o otra.
