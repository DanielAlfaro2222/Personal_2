Border

El border es la linea que rodea a la caja, tambien esta propiedad es un shorthand, que junto a border, border-width, border-color y border-style, agrupan cada una todos los lados de la caja.

Ejemplo

Cada valor corresponde a un lado de la caja, el primer valor corresponde al lado superior o border-top, el segundo corresponde al lado derecho o border-right, el tercero corresponde al lado inferior o border-bottom y el cuarto corresponde al lado izquierdo o border-left. 
				
					/* Border solo */

					.box-4 {
						border: 1em 2em 3em 4em;
					}
					
					/* Border width */

					.box-4 {
						border-width: 1em 2em 3em 4em;
					}
					
					/* Border style */

					.box-4 {
						border-style: dotted double groove ridge;
					}
					
					/* Border color */

					.box-4 {
						border-color: white blue green yellow;
					}
