Position fixed

La position fixed es la que permite que un elemento se quede fijado en el documento, sin importar si realiza o no scroll, el elemento se quedara todo el tiempo en esa misma posicion.

Como el elemento se queda fijo en esa posicion, lo que hace la propiedad fixed es quitarle el espacio en el flujo del documento HTML y se posicionara respecto al viewport o body, ademas para ubicarlo tenemos cuatro propiedades mas, una vez el elemento este usando el position fixed.

Si el elemento no tiene un tamaño declarado, su tamaño sera igual al tamaño de su contenido

Propiedades para ubicar o mover el elemento con position fixed

  • top => Se usa para mover el elemento hacia arriba en relacion al body o viewport, acepta valores positivos o negativos.
  • left => Se usa para mover el elemento hacia la izquierda en relacion al body o viewport, acepta valores positivos o negativos.
  • right => Se usa para mover el elemento hacia la derecha en relacion al body o viewport, acepta valores positivos o negativos.
  • bottom => Se usa para mover el elemento hacia abajo en relacion al body o viewport, acepta valores positivos o negativos.
