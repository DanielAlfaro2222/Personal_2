Propiedad z-index

La propiedad z-index se usa para indicar el orden de apilamiento de dos o mas elementos en el eje z, entre mas valor de z-index tenga un elemento, mas arriba se posicionara, respecto a los demas.

Consideraciones de z-index

  • Esta propiedad acepta solo valores enteros, ademas pueden ser valores positivos o negativos.
  • Se recomienda usar valores no consecutivos, pueden ser de 5 en 5, de 10 en 10, de 100 en 100, etc...
  • Si queremos que un contendor padre se posiciones encima de un contenedor hijo, deberemos tener en cuente dos condiciones para que esto funcione, primero el contenedor padre no debe tener un z-index declarado, y segundo el contenedor hijo debe tener un z-index negativo.
