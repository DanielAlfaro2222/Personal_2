Metodologia BEM

Una metodologia en CSS son una serie de consejos para estructurar nuestras clases de una manera sencilla, estable y reutilizable.

BEM es una de las metodologias mas utilizadas en el mundo.

Su nombre viene de las siglas Block, Element, Modifier.

  • Block (Bloque) => Es cualquier elemento autonomo y aislado de nuestro documento. Por ejemplo un menu, una galeria, un formulario, etc....
  • Element (Eleemnto) => Es cada uno de los elementos del bloque. Por ejemplo un link, una foto, un campo de texto, etc...
  • Modifier (Modificador) => Es cuando un bloque se repite en otro lugar de nuestra web y necesita una modificacion. Por ejemplo un color, un texto, una fuente, etc...

Como se nombran los bloques

Los bloques se nombran con una sola palabra o con dos separadas por un guion, por si llegasen a existir conflictos de estilos.

Ejemplo

.menu / .main-menu
.gallery / .main-gallery

Como se nombran los elementos

Para nombrar los elementos se usa el nombre del bloque al que pertenecen primero, luego dos rayas al piso y el nombre del elemento

Ejemplo

.menu__link / .main-menu__link
.gallery__img / .main-gallery__img

Como se nombran los modificadores

Para nombrar los modificadores primero se usa el nombre del bloque al que pertenecen, luego dos guiones y el nombre del modificador.

Ejemplo

.menu--color / .main-menu--color
.gallery--color / .main-gallery--color
