Prefijos

En CSS3 las propiedades nuevas o que estan en experimentos, como no son compatibles con los navegadores, deben usar prefijos para que sean compatibles, para esto usaremos un plugin de visual estudio code llamado autoprefixed, el cual se encargara de agregar los prefijos necesarios si una propiedad los necesita.

No tenemos que aprendernos los prefijos, ya que son muchos, es mejor dejar que el plugin se encargue.
