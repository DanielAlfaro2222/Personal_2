Propiedad display

La propiedad display se usa para indicar el comportamiento de una caja, con respecto a las demas cajas a su alrededor

Tipos de display mas utilizados

  • inline => se usa para indicarle al elemento, que se va a comportar como un elemento en linea.Ademas, los margenes y padding en el eje vertical, no afectaran a las demas cajas, unicamente los margenes y padding en el eje horizontal.
  • block => se usa para indicarle al elemento, que se va a comportar como un elemento en bloque.
  • inline-block => Se usa para indicarle al elemento, que se va a comportar como un elemento en linea, pero este si podra usar las propiedades que tienen los elementos en bloque, en resumen es una combinacion de ambos.
  • none => Se usa para ocultar elementos, pero de igual forma la pagina los cargara, si un elemento tiene este display, tambien perdera su espacio en el flujo del html.
  • grid => mas adelante veremos para que se usa.
  • flex => mas adelante veremos para que se usa.
