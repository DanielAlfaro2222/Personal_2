Estructura basica de CSS

  • Regla => Es todo el conjunto del bloque de codigo(selector, llaves, propiedad-valor
  • Selector => Es el nombre de la etiqueta HTML que queremos modificar
  • Propiedad => Es el nombre de la propiedad que vamos a utilizar, se debe usar al final punto y coma para separar cada propiedad, ademas podremos usar varias propiedades.
  • Valor => Es el valor de la propiedad la cual vamos a aplicar.

Ejemplo de una regla de CSS

            
                Selector{
                    propiedad: valor;
                }
