Resetear los estilos de los navegadores con normalize CSS

Los navegadores por defecto, manejan una hoja de estilos predeterminados, entonces, para resetear los estilos que trae por defecto el navegador, vamos a usar normalize CSS.

    Consejos

  • Para evitar conflictos entre normalize CSS y nuestros estilos CSS, primero colocaremos el normalize y luego todos nuestros estilos para evitar conflictos o que los conflictos se resuelvan con cascada
  • En el archivo de Normalize CSS agregaremos una nueva regla con un selector universal y la propiedad box-sizing: border-box. Esta regla nos permitira que si le agregamos padding por ejemplo a un div, el padding no se desborde, sino que se calcule para que el div siga teniendo el mismo tama√±o.
