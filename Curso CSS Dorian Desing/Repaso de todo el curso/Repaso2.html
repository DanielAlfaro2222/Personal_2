Formas de insertar estilos en HTML5

  • Usando la etiqueta style en el head de nuestra pagina: Esta forma se recomienda solamente si es una sola pagina, ya que si tenemos varias paginas, lo recomendables es que la estructura y los estilos de las paginas esten separados.
  • Usando estilos en linea usando el atributo style: Esta forma se recomienda usar si y solamente si cuando tengamos que editar los estilos en tiempo real con javascript, del resto no se recomienda usar.
  • Usando clases: Esta es la forma mas recomendable para darle estilos a nuestro HTML5.
  • Usando @import: Esta forma ni por el putas hay que usarla, ya que si llegase a fallar, el resto de los estilos no cargaran.

Especificidad, herencia y cascada

Especificidad

La especificidad es el peso o valor que tiene un selector, si tenemos un conflicto entre dos reglas que se deberian aplicar a un mismo elemento, ganara la que mas especificidad(valor) tenga.

Valores de especificidad

  1. Selector de etiqueta........1
  2. Selector de clase...........10
  3. Selector de id..............100
  4. Estilo en linea.............1000
  5. !important..................le gana a todo

Herencia

La herencia es cuando un elemento hijo hereda estilos de un elemento padre, solo hereda algunos estilos, existe tres valores el primero es el inherit(este fuerza la herencia), initial(este va a tomar el color por defecto que trae el elemento, no lo estilos del padre) y unset(este puede hacer dos cosas, todo depende de la ubicacion, si esta dentro de un elemento va a heredar los estilos del elemento padre y si esta por fuera, va a tomar los estilos por defecto del propio elemento).

Metodologia BEM

La metodologia BEM es una forma de nombrar las clases en HTML5 para estructurar de una manera eficiente el codigo en CSS. las siglas de BEM significan B(block), E(element), M(modifier).

Como se nombran las clases

  • Bloques => Los bloques se nombran con una palabra, dos palabras o mas separadas por un guion. Ejemplo class = "main-menu"
  • Elementos => Los elementos se nombran primero se pone el bloque al que pertenecen, luego dos rayas al piso y el nombre del elemento.Por ejemplo class = "main-menu__img"
  • Modifier => Las modificaciones se nombran primero el bloque al que pertecen, luego, dos rayas el nombre del elemento, y luego dos guiones el nombre de la modificacion. Por ejemplo class = "main-menu__img--active"

Box model

En CSS3 cada elemento es visto como una caja, cada caja se compone de un contenido, un padding, un border y un margin (este no hace crecer el tamaño real de la caja).

  • Content: Es el contenido principal de la caja.
  • Padding: Es la separacion del contenido y el border en la caja, esta propiedad hace crecer el tamaño real de la caja.
  • Border: Es la separacion entre padding y el margin, esta propiedad hace crecer el tamaño real de la caja.
  • Margin: Es la separacion entre una caja, con las demas cajas, esta propiedad no hace crecer el tamaño real de la caja.
  • Outline: Se usa para ver la ubicacion de una caja, no hace parte del box model, ademas que no modifica el tamaño real de la caja.

Propiedades margin y padding

Margin

El margin es la sepacion de una caja, con las otras cajas a su alrededor.Ademas, el margin es una propiedad shorthand, ya que con solo esta propiedad podremos modificar varias cosas, sin necesidad de usar las propiedades margin-top, margin-bottom, margin-left, margin-right, usando la propiedad margin podremos modificar estas cuatro propiedades, el primer valor correspondera al margin-top, y los siguientes valores corresponderan deacuerdo a las manecillas del rejo, osea el primer valor corresponde al margin top, el segundo valor corresponde al margin right, el tercer valor corresponde al margin bottom y el cuarto valor corresponde al margin left.

Por ejemplo

margin: 10px(margin-top) 5px(margin-right) 2px(margin-bottom) 3px(margin-left);

Padding

El padding es la separacion del contenido con el borde de la caja, esta propiedad tambien es un shorthand, ya que con esa sola propiedad podremos modificar los cuatro lados de la caja, el primer valor correspondera al padding superior, el segundo valor correspondera al padding derecho, el tercer valor correspondera al padding inferior, el cuarto valor correspondera al padding izquierdo.

Ejemplo

padding: 1em(padding-top) 2em(padding-right) 3em(padding-bottom) 4em(padding-left);
