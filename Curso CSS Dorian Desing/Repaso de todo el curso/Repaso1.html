Como insertar CSS en HTML

En HTML5 podremos insertar CSS, de diferentes maneras.

  • En linea: Se usa dentro de la misma etiqueta HTML5, usando el atributo style, esto solo se usa cuando hay cambios en vivo desde javascript, del resto no se recomienda usar para nada.
  • Usando la etiqueta style: En el head de nuestro HTML5, esta manera solo se utiliza cuando tenemos una sola pagina o para practicar, pero no es recomendable cuando tenemos varias paginas que comparten estilos, ya que nos tocaria tener lo mismos en todas las paginas.
  • Insertar un archivo externo: En el head de nuestro HTML5, usando la etiqueta link, y el atributo href pondremos la direccion o la ubicacion del archivo, de esta manera es la mas recomendable de usar.
  • A traves de @import: En el head de nuestro HTML, en la etiqueta style, usaremos el @import para importar nuestros archivos de CSS, pero no es recomendable de usar ya que si llegase a fallar, el resto de la pagina no cargara.

Composicion de CSS

  • Regla: Es todo el bloque de codigo, maneja su selector y las propiedades que manejen.
  • Selector: Es el nombre de la etiqueta HTML, a la cual le vamos a aplicar el estilo.
  • Propiedad: Es la funcion o el nombre del estilo que vamos a aplicar, maneja nombre de la propiedad y un valor, cada propiedad se separa con punto y coma para indicar donde termina una propiedad y donde empieza otra.

    • Ejemplo de una regla en CSS3

                    
                    Selector{
                        propiedad: valor;
                    }

Metodologia BEM

La metodologia BEM es una forma de nombrar las clases en HTML, para tener una mejor forma de estructurar el codigo CSS3 para asi poder reutilizarlo, usar la menor cantidad posible de lineas de codigo y poder arreglar errores de una mejor manera.

Definicion de las siglas

  • Block: Es nombre de la etiqueta que empieza el bloque, por ejemplo si tenemos un div con varios parrafos dentro, la etiqueta de bloque seria el div.Para nombrarla tendremos dos opciones, primera si el bloque principal, le pondremos main-container, o solo container, ambas opciones son validas.
  • Element: Es un elemento que pertenece a un bloque. Para nombrar un elemento de bloque, lo que haremos es primero poner el nombre del bloque al que pertenece, luego dos rayas al piso y el nombre del elemento.
  • Modifier: Es un modificador de un elemento, se usa para cuando tenemos dos elementos en nuestra web iguales y queremos modificarle los estilos a uno de los dos elementos, con esto logramos que no tengamos que estar repitiendo codigo innecesario, solo agregarle una nueva clase al elemento y en CSS usar esa clase para modificarle los estilos.Para nombrar la clase con el modifier primero ponemos el bloque al que pertenece la etiqueta y luego dos guiones y el nombre de la modificacion.

    Ejemplo

                            
                            -------------------HTML5--------------------
                            <div class = "main-container">
                                <a class = "main-container__link" href = "#">
                                    Enlace 1
                                </a>
                                <a class = "main-container__link main-container--letter" href = "#">
                                    Enlace 2
                                </a>
                            </div>

                            -------------------CSS3--------------------
                            .main-container__link{
                                color: red;
                                text-decoration: none;
                                font-size: 20px;
                                font-family: cursive;
                            }

                            .main-container--letter{
                                letter-spacing: 5px;
                                padding: 5px;
                                background-color: blue;
                            }

Resetear los estilos de los navegadores

Los navegadores por defecto ya traen una hoja de estilos incorporada, para lo cual nosotros tendremos que resetear los estilos que el navegador o los navegadores ya traen por defecto, para eso usaremos normalize.CSS el cual es una hoja de estilos para poder resetear los estilos por defecto de los navegadores, la recomendacion es que en la parte del head de nuestro sitio web, debemos insertar primero el normalize y luego nuestros estilos, para que no existan conflictos entre ambos estilos.

Ejemplo de como se deberian insertar los estilos

                
                    <head>
                        <link href = "normalize.css" rel = "stylesheet">
                        <link href = "styles.css" rel = "stylesheet">
                    </head>

Prefijos

En CSS los prefijos son unos complementos de los provedores o desarrolladores a las propiedades para que sean compatibles con el navegador, no todas las etiquetas manejan prefijos, pero si las que estan en fase experimental.

Ejemplo de una etiqueta con prefijos

                
                    body {
                        -webkit-appearance: initial;
                           -moz-appearance: initial;
                                appearance: initial;
                    }

Especificidad, Herencia y Cascada

Estos tres temas son importantes conocerlos, ya que cuando se presenten conflictos entre los estilos, usaremos alguna de estas tres formas para resolverlos.

Especificidad

La especificidad es el valor que tiene el selector de la regla, si un elemento tiene dos o mas reglas, en las cuales hay propiedades que tienen conflictos, el navegador va a aplicar las propiedades que tengan un selector con mas especificidad, mas no va a sobreescribir toda la regla, solo va a sobreescribir las propiedades en conflicto.

Orden de especificidad

  1. Etiqueta----------------------0,0,0,1
  2. clases y pseudoclases---------0,0,1,0
  3. ID----------------------------0,1,0,0
  4. Estilo en linea---------------1,0,0,0
  5. Important---------------------Le gana a cualquiera(no usarlo)

Ejemplo de Especificidad

                    
                        -------------------HTML5--------------------
                        <a href="#" class = "link" id = "link">Enlace</a>

                        -------------------CSS3---------------------
                        #link {
                            color: red; // Se aplicara esta propiedad ya que tiene mas especificidad, tambien se aplicaran el resto de propiedades ya que no estan entrado en conflicto.
                            text-decoration: none;
                        }
                        .link {
                            font-size: 70px;
                            color: blue;
                            font-family: cursive;
                        }

Herencia

La herencia es como su mismo nombre lo dice, la herencia de algunos estilos del elemento padre al elemento hijo, por ejemplo si dentro de un parrafo tenemos un texto enfatizado, y al parrrafo le aplicamos un color, el texto enfatizado lo heredara. Hay casos en los que las propiedades no se heredan, por ejemplo en los enlaces, estos no heredan estilos. Tenemos tres valores de propiedades initial(no fuerza la herencia, sino que mas bien se toma el valor por defecto del elemento), inherit(Fuerza la herencia, lo que hace que tome el valor del elemento padre), unset(Este valor basicamente puede hacer dos cosas, dependiendo de como este ubicadas las etiquetas si esta dentro de otra etiqueta lo que va a hacer es reestablecer la propiedad a la que heredo y no a los valores iniciales, y si esta fuera de una etiqueta lo que va a hacer es reestablecer la propiedad a los valores iniciales).

Ejemplo de herencia

                    
                        -------------------HTML5--------------------
                        <p class = "paragraph">
                            Lorem, ipsum dolor sit adipisicing elit. Saepe porro <em>cupiditate illum</em> harum tempore. 
                        </p>

                        -------------------CSS3---------------------
                        .paragraph{
                            color: green;
                        }

Cascada

La cascada es cuando un elemento(selector, clase o id), tiene mas de una regla de estilo para aplicar, lo que hace el navegador es aplicar la ultima regla que tenga ese elemento, sin embargo, no se reescribe toda la regla, mas bien se reescribe las propiedades que entran en conflicto.A esto se le llama cascada.

Ejemplo de cascada

                    
                        -------------------HTML5--------------------
                        <a href = "#" class = "main-link">Enlace</a>
                        
                        -------------------CSS3---------------------
                        .main-link{
                            color: blue;
                           font-size: 20px;
                           font-family: verdana;
                        }
                        .main-link{
                            color: white;
                        }
                        .main-link{
                            color: red; //Por cascada se va a aplicar esta propiedad y el resto de colores los va a ignorar.
                        }

Orden en como se resuelven los conflictos en CSS3

  1. Important
  2. Especificidad
  3. Cascada

Tipos de selectores

En CSS existen diferentes selectores.

  1. Selector de etiqueta: Este es el selector mas basico, es cuando usamos el nombre de la etiqueta a la cual queremos afectar, no es recomendable usar ya que afecta de manera global a la misma etiqueta en toda la pagina.

    Ejemplo de selector de etiqueta

                            
                            h2{
                                color: red;
                                font-size: 20px;
                                padding: 5px;
                            }
  2. Selector de id: Este tipo de selector no es recomendable usarlo para darle estilos a una etiqueta, se recomienda usar para identificar una etiqueta en javascript para darle alguna funcionalidad, al ser un identificador, este no se puede repetir, debe ser unico.

    Ejemplo selector de id

                            
                            #parrafo{
                                color: blue;
                                font-family: verdana;
                                padding: 5px;
                            }
  3. Selector de clase: Este tipo de selector es el mas usado y el mas recomendado para darle estilos a una etiqueta en CSS, en este tipo de selector no hay problema si se repite, lo podremos usar cuantas veces queramos, dentro de la etiqueta HTML usaremos el atributo class y le daremos un valor, este valor luego lo llamaremos en CSS de la siguiente forma:

    Ejemplo de selector de clase

                            
                            .main-menu{
                                background-color: blue;
                                color: white;
                                font-size: 20px;
                            }
  4. Selector de atributo:
    • Selector de atributo

      Ejemplo de selector de atributo

                                      
                                    -------------------HTML5--------------------
                                    <a href = "#">Enlace</a>

                                    -------------------CSS3---------------------
                                    [href]{
                                        color: red;
                                    }
    • Selector de atributo y valor

      Ejemplo de selector de atributo y valor

                                      
                                    -------------------HTML5--------------------
                                    <a href = "#">Enlace</a>

                                    -------------------CSS3---------------------
                                    [href = "#"]{
                                        color: red;
                                    }
    • Selector de atributo y valor por lo menos una vez

      Ejemplo de selector de atributo y valor por lo menos una vez

                                      
                                    -------------------HTML5--------------------
                                    <a class = "uno-dos">Enlace</a>

                                    -------------------CSS3---------------------
                                    [class ‚Åì= "dos"]{
                                        color: red;
                                    }
    • Selector que tengan exactamente ese atributo y ese valor o que por lo menos empiece por ese valor separado por un guion

      Ejemplo de selector que tenga exactamente ese atributo y ese valor 

                                      
                                    -------------------HTML5--------------------
                                    <a class = "dos">Enlace</a>

                                    -------------------CSS3---------------------
                                    [class |= "dos"]{
                                        color: red;
                                    }
    • Selector que tenga ese atributo y empiecen por ese valor

      Ejemplo de selector que tenga ese atributo y empiece por ese valor

  5. Selectores combinadores:
    • Selector de hermano adyacente: Esto lo que va a hacer es aplicar estilos al elemento que este justo debajo de el.

      Ejemplo selector de hermano adyacente

                                      
                                    -------------------HTML5--------------------
                                    <h1>Titulo</h1>
                                    <h2>Subtitulo</h2>

                                    -------------------CSS3---------------------
                                    h1 + h2{
                                        color: blue; //se va a aplicar esta propiedad a la etiqueta h2
                                    }
    • Selector de hermano general: Esto lo que va a hacer es aplicar estilos a todas las etiquetas que esten debajo de la etiqueta padre, esto se hace de arriba hacia abajo, si queremos realizarlo al reves tendremos que usar javascript.

      Ejemplo selector de hermano general

                                      
                                    -------------------HTML5--------------------
                                    <h1>Titulo padre</h1>
                                    <h2>Sutitulo hijo</h2>
                                    <h2>Sutitulo hijo</h2>

                                    -------------------CSS3---------------------
                                    h1 ~ h2{
                                        color: red;
                                    }
    • Selector de hijo directo: Esto lo que va a realizar es aplicar estilo a una etiqueta que esta debajo de una etiqueta especifica y no de otra.

      Ejemplo selector de hijo directo

                                      
                                    -------------------HTML5--------------------
                                    <h1>Titulo padre</h1>
                                    <p>Parrafo</p>

                                    -------------------CSS3---------------------
                                    h1 > p{
                                        color: red;
                                    }
    • Selector de hijo descendente: Esto lo que va a indicar es que se le aplicara los estilos a una etiqueta que esta dentro de otra, solo se aplicara los estilos si se cumple esta condicion.

      Ejemplo de hijo descendiente

                                      
                                    -------------------HTML5--------------------
                                    <div>
                                        Contenedor
                                        <h2>Sutitulo hijo</h2>
                                        <h2>Sutitulo hijo</h2>
                                    </div>

                                    -------------------CSS3---------------------
                                    div h2{
                                        color: red;
                                    }
  6. Selector universal: Este tipo de selector se usa para aplicar estilos a todo el documento HTML, no es muy recomendable usar este tipo de selector.

    Ejemplo de selector universal

                            
                            *{
                                color: red;
                                letter-spacing: 5px;
                            }
