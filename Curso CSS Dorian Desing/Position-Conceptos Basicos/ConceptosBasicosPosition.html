Conceptos Basicos Position

Position en pocas palabras, es la posicion de un elemento en un documento HTML.

Tipos de position

A su vez tenemos cuatro tipos de posicionamiento en un documento HTML5.

  • Static (por defecto) => Si usamos este, significara que el elemento no esta posicionado, para que se considere posicionado debemos usar los demas position
  • Relative
  • Sticky
  • Absolute o Fixed

Propiedades de position

Las propiedades top, left, right y bottom determinan la ubicacion final del elemento en el documento.

  • Top => Sirve para poder mover el elemento en relacion a la parte superior.
  • Left => Sirve para poder mover el elemento en relacion a la parte izquierda.
  • Right => Sirve para poder mover el elemento en relacion a la parte derecha
  • Bottom => Sirve para poder mover el elemento en relacion a la parte inferior.
  • Z-index => Sirve para mover el elemento en el eje z, ademas si tenemos dos elementos en la misma ubicacion, podremos con esta propiedad podremos decidir, cual elemento estara encima y cual estara al fondo.

Espacio en el documento

El espacio en el documento, es en pocas palabras es el espacio que ocupa un elemento en el documento.
