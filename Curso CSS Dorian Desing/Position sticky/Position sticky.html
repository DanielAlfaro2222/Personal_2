Position sticky

El position sticky, es una combinacion entre position relative y position fixed, este tipo de position, si respeta el espacio que ocupa el elemento en el flujo del HTML, ademas para que funcione bien el sticky, al elemento que le vamos a aplicar, el position, tiene que estar por fuera de un contenedor, si esta adentro, el elemento se quedara pegado hasta el final del tamaño del contenedor y no del tamaño final de la pagina.

Si el elemento con position sticky, no tiene un tamaño o dimension declarado, su tamaño dependera del tamaño de su contenido.

Este tipo de position se usa principalmente para crear un menu o un pie de pagina, que sin importar el scroll que se haga, el elemento siempre va a conservar su posicion.

Este tipo de position primero funcionara como un postion relative, y luego de que se alcance el tope declarado, funcionara como un postion fixed.

Propiedades de position sticky

El position sticky, acepta cuatro propiedades, las cuales son top, left, right, bottom, de estas cuatro propiedades, las que mas peso tienen son el top y el left.

  • top => Se usa para mover el elemento hacia arriba.
  • left => Se usa para mover el elemento hacia el lado izquierdo.
  • right => Se usa para mover el elemento hacia el lado derecho.
  • bottom => Se usa para mover el elemento hacia abajo.
