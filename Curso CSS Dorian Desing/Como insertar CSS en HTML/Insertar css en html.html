Maneras de insertar CSS en HTML5

  1. En la cabecera: a traves de la etiqueta <style>: En caso de que tengamos una sola pagina, entonces ahi si podremos usar este metodo, pero si tenemos varias paginas que comparten estilos no es recomendable usar este metodo ya que tendriamos que colocar en cada pagina el mismo codigo.
  2. En linea dentro de la etiqueta con el atributo style: Se usa cuando se necesita cambiar los estilos en tiempo real desde javascript
  3. Asociando una hoja de estilos externa a nuestro documento HTML: Esta es la forma mas usada y mas recomendada.
  4. A traves de @import dentro de la etiqueta <style>: Esta es la peor forma de insertar CSS en HTML5 ya que si llegase a fallar este import, el resto de la pagina no cargara.
