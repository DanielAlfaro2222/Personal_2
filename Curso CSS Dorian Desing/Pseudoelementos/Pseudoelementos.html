Pseudoelementos

Los pseudoelementos se usan para aplicar estilos a partes especificas de los elementos.

Tipos de pseudoelementos mas utilizados

  • ::before => Se usa para aplicar o dibujar cosas antes del elemento, ademas este pseudoelemento, es dibujado de manera virtual, si intentamos seleccionar el elemento con el mouse, no nos dejara seleccionarlo.Ademas este pseudoelemento para funcionar debe tener la propiedad contenent, no importa si esta vacia, sin esa propiedad el before no funcionara, el contenent se usa para insertar letras o simbolos.
  • ::after => Se usa para aplicar o dibujar cosas despues del elemento, ademas este pseudoelemento, es dibujado de manera virtual, si intentamos seleccionar el elemento con el mouse, no nos dejara seleccionarlo.Ademas este pseudoelemento para funcionar debe tener la propiedad contenent, no importa si esta vacia, sin esa propiedad el after no funcionara, el contenent se usa para insertar letras o simbolos.
  • ::first-letter => Se usa para aplicar estilos a la primera letra de una palabra, solo funciona en elementos en bloque.
  • ::first-line => Se usa para aplicar estilos a la primera linea de un texto, esto funciona dinamicamente, si la pantalla es muy larga o peque√±a se va a seguir aplicando estos estilos a la primera linea del texto, solo funciona en elementos en bloque.
  • ::selection => Se usa para aplicar estilos a la parte donde el usuario selecciona un texto con el mouse, este pseudoelementos actualemente esta en obsoleto o no se usa con frecuencia.
